Note you must have ActiveSync (WIN XP) or Windows Mobile Device Center (WIN 7) to connect the handheld to your PC.

Recommend Install order:
1) Compact .NET Framework (NETCFv35.wm.armv4i.cab)
2) Compact SQL (sqlce.phone.wce5.armv4i.CAB)
3) Cart Manager (Cart Manager Installer.CAB)
4) MyMobiler -Freeware (MyMobiler.exe), used to view handheld screen on PC.
